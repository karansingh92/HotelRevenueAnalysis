# HotelRevenueAnalysis


1. Project Title

Hotel Performance Analysis & Revenue Management Dashboard

2. Short Description / Purpose

This project is an interactive Power BI dashboard designed to analyze and monitor hotel performance across multiple business dimensions, including revenue, occupancy, guest behavior, booking channels, and cancellation trends.

The purpose of this dashboard is to simulate a real-world hotel analytics system that supports revenue management, operational efficiency, and strategic decision-making. It provides a comprehensive overview of performance metrics while enabling deeper analysis through interactive visualizations.

3. Tech Stack

1. Microsoft Power BI Desktop – Data visualization and dashboard development
2. DAX (Data Analysis Expressions) – KPI and business metric calculations
3. Data Modeling (Star Schema Architecture) – Relationship and performance optimization
4. Power BI Service (Conceptual Deployment) – Role-based access and usage tracking

 4. Data Information

The dataset used in this project consists of structured hotel booking data, including:

1. Booking details (Status: Completed, Cancelled, No-Show, Nights, Revenue, Channel)
2. Customer information (Customer ID, Nationality, Segment)
3. Room information (Room Category)
4. Hotel branch details
5. Date dimension (Year, Month, Quarter, Season classification)

5. Features / Highlights

1.Revenue analysis with trend and forecast visualization
2.Occupancy %, ADR, and RevPAR calculation
3.Guest segmentation and lifetime value analysis
4.Booking channel performance evaluation
5.Cancellation and No-Show impact assessment
6.Refund and Net Revenue modeling
7.Seasonal and branch-level performance comparison
8.Interactive navigation using bookmarks
9.Professional dark-themed executive dashboard layout
